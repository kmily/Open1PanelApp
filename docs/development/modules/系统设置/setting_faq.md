# 系统设置模块FAQ

## 常见问题

### 1. 如何修改面板端口？

**操作步骤**:
1. 进入系统设置 > 面板设置
2. 找到"端口设置"选项
3. 输入新端口号（1024-65535）
4. 保存设置
5. 重启面板服务

**注意**: 修改端口后需要重新登录

### 2. 如何启用双因素认证？

**操作步骤**:
1. 进入系统设置 > 安全设置
2. 找到"双因素认证"选项
3. 点击"启用"按钮
4. 使用认证器APP扫描二维码
5. 输入验证码确认启用

### 3. IP白名单如何配置？

**配置方法**:
1. 进入系统设置 > 安全设置
2. 找到"IP白名单"选项
3. 点击"添加IP"
4. 输入IP地址或CIDR格式
5. 保存配置

**格式示例**:
- 单个IP: 192.168.1.100
- IP范围: 192.168.1.0/24
- 多个IP: 192.168.1.100,192.168.1.101

### 4. 如何创建系统快照？

**创建步骤**:
1. 进入系统设置 > 快照管理
2. 点击"创建快照"按钮
3. 输入快照描述
4. 选择快照内容
5. 确认创建

### 5. 快照恢复会覆盖哪些内容？

**恢复范围**:
- 系统配置文件
- 应用配置
- 数据库配置
- 用户设置

**不会覆盖**:
- 用户数据
- 应用数据
- 数据库数据

### 6. 如何配置邮件通知？

**配置步骤**:
1. 进入系统设置 > 通知设置
2. 点击"添加通知渠道"
3. 选择"邮件"类型
4. 配置SMTP服务器信息
5. 测试发送
6. 保存配置

### 7. 设置保存后不生效怎么办？

**排查步骤**:
1. 检查是否有错误提示
2. 验证设置值格式正确
3. 检查服务是否需要重启
4. 查看系统日志
5. 尝试重置设置

### 8. 如何导出系统配置？

**导出方法**:
1. 进入系统设置 > 快照管理
2. 点击"导出快照"
3. 选择需要导出的配置项
4. 选择导出格式
5. 下载配置文件

### 9. 会话超时时间如何设置？

**设置方法**:
1. 进入系统设置 > 安全设置
2. 找到"会话设置"选项
3. 修改超时时间（分钟）
4. 保存设置

**建议**: 生产环境建议设置30-60分钟

### 10. 如何重置所有设置？

**重置方法**:
1. 进入系统设置 > 面板设置
2. 点击"重置设置"按钮
3. 确认重置操作
4. 系统将恢复默认设置

**警告**: 重置后所有自定义设置将丢失

## 故障排查指南

### 设置保存失败
```
错误: 设置保存失败
排查:
1. 检查设置值格式
2. 验证权限是否足够
3. 检查存储空间
4. 查看系统日志
```

### 双因素认证问题
```
错误: 验证码无效
排查:
1. 检查手机时间是否正确
2. 重新扫描二维码
3. 确认认证器APP正常
4. 联系管理员重置
```

### 快照创建失败
```
错误: 快照创建失败
排查:
1. 检查存储空间
2. 验证文件权限
3. 检查系统负载
4. 查看详细错误日志
```
