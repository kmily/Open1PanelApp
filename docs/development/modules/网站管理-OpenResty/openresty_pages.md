# 网站模块 - OpenResty 子模块页面原型与交互设计

## 页面清单

1. OpenResty 概览页
2. 配置编辑页
3. HTTPS 默认配置页
4. 模块管理页
5. 构建任务页

## 页面结构

### 1. 概览页

- 顶部状态卡片: 运行状态、版本、Nginx版本
- 指标卡片: 连接数、请求数、读写等待
- 路径信息: 配置路径、日志路径、错误日志路径
- 操作入口: 配置编辑、HTTPS、模块管理、构建

交互说明:
- 页面加载时请求状态与服务状态
- 点击刷新时重新拉取状态
- 运行异常时提示与错误引导

### 2. 配置编辑页

- 左侧导航: 全量配置、局部配置、历史版本
- 编辑器区域: 语法高亮、只读切换
- 操作区: 校验、保存、回滚、下载、上传

交互说明:
- 初次进入拉取当前配置
- 提交前执行语法校验与差异预览
- 提交成功后触发服务重载

### 3. HTTPS 默认配置页

- 开关控制: 默认HTTPS启用/停用
- 证书信息: 当前证书、有效期、关联域名
- 影响提示: 对新站点默认生效范围

交互说明:
- 启用前检查证书是否存在
- 操作完成后刷新状态并提示

### 4. 模块管理页

- 模块列表: 模块名称、版本、启用状态
- 批量操作: 启用/禁用、更新
- 依赖提示: 冲突模块提示

交互说明:
- 勾选模块后提交更新
- 更新完成刷新状态

### 5. 构建任务页

- 任务状态: 进行中/成功/失败
- 构建参数: 目标版本、模块集合
- 日志输出: 构建日志与错误提示

交互说明:
- 触发构建后进入日志跟踪
- 失败时提供重试入口
