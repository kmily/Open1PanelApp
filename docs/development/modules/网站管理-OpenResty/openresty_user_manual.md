# 网站模块 - OpenResty 子模块用户操作手册

## 概览

本模块用于管理OpenResty的运行状态、配置、HTTPS默认配置与模块扩展能力，适用于 Open1PanelApp。

## 常用操作

### 查看运行状态

1. 打开OpenResty概览页
2. 查看运行状态、版本与连接指标

### 修改配置

1. 进入配置编辑页
2. 修改后执行校验与保存
3. 提交并观察服务状态

### 设置默认HTTPS

1. 进入HTTPS默认配置页
2. 选择启用或禁用
3. 确保证书配置完整

### 管理模块

1. 进入模块管理页
2. 勾选目标模块并提交更新
3. 等待更新完成并刷新状态
