# 系统分组模块索引

## 模块定位

系统分组模块是Open1PanelApp的P2工具类模块，提供资源的分组管理能力，支持将应用、网站、数据库等资源按业务或环境进行分类管理，便于批量操作和统一监控。

## 子模块结构

| 子模块 | 端点数 | API客户端 | 说明 |
|--------|--------|-----------|------|
| **分组管理** | 4 | system_group_v2.dart | 分组创建、编辑、删除 |
| **资源关联** | 2 | system_group_v2.dart | 资源与分组的关联管理 |
| **分组查询** | 2 | system_group_v2.dart | 按分组查询资源 |

## 与其他模块的关系

- **应用管理**: 应用资源分组
- **网站管理**: 网站资源分组
- **数据库管理**: 数据库资源分组
- **监控管理**: 按分组聚合监控数据
