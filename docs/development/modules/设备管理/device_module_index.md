# 设备管理模块索引

## 模块定位

设备管理模块是Open1PanelApp的P2工具类模块，提供服务器硬件设备的监控和管理能力，包括CPU、内存、磁盘、网络等设备的状态查看和基础管理。

## 子模块结构

| 子模块 | 端点数 | API客户端 | 说明 |
|--------|--------|-----------|------|
| **设备列表** | 4 | host_v2.dart | 设备信息列表展示 |
| **设备详情** | 4 | host_v2.dart | 单个设备详细信息 |
| **设备监控** | 2 | host_v2.dart | 设备状态实时监控 |
| **设备操作** | 2 | host_v2.dart | 设备基础操作管理 |

## 后续规划方向

### 短期目标
- 完善设备信息展示页面
- 实现设备状态实时监控
- 添加设备告警功能

### 中期目标
- 支持设备性能趋势图表
- 实现设备健康评分
- 添加设备对比功能

### 长期目标
- 支持多服务器设备管理
- 实现设备资源规划
- 提供设备生命周期管理

## 与其他模块的关系

- **主机管理**: 共享主机基本信息
- **监控管理**: 提供设备监控数据
- **日志管理**: 记录设备操作日志
- **工具箱**: 磁盘管理联动
