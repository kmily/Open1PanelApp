# 网站域名管理模块索引

## 模块定位

网站域名管理模块是Open1PanelApp的P1高价值扩展模块，提供网站域名的解析管理能力，包括DNS记录配置、域名绑定、域名解析状态监控等，是网站运行的基础配置模块。

## 子模块结构

| 子模块 | 端点数 | API客户端 | 说明 |
|--------|--------|-----------|------|
| **域名绑定** | 4 | website_v2.dart | 网站域名绑定和解绑 |
| **DNS管理** | 4 | website_v2.dart | DNS记录配置和管理 |

## 后续规划方向

### 短期目标
- 完善域名绑定功能
- 实现DNS记录管理
- 添加域名解析状态监控

### 中期目标
- 支持多DNS服务商
- 实现域名自动解析
- 添加域名健康检查

### 长期目标
- 支持域名批量管理
- 实现域名监控告警
- 提供域名安全评估

## 与其他模块的关系

- **网站管理**: 域名与网站关联
- **SSL证书管理**: 域名证书绑定
- **监控管理**: 域名解析监控
- **日志管理**: 域名操作日志
