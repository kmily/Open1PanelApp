# 网站域名管理模块FAQ

## 常见问题

### 1. 如何绑定域名到网站？

**绑定步骤**:
1. 进入网站域名管理页面
2. 点击"添加域名"
3. 输入域名地址
4. 配置域名参数
5. 验证DNS解析
6. 完成绑定

### 2. 域名绑定失败怎么办？

**排查步骤**:
1. 检查域名格式是否正确
2. 验证DNS解析是否生效
3. 检查域名是否已被绑定
4. 确认操作权限

### 3. 如何配置DNS记录？

**配置方法**:
1. 进入DNS管理页面
2. 点击"添加记录"
3. 选择记录类型
4. 输入记录值
5. 设置TTL
6. 保存配置

### 4. 支持哪些DNS记录类型？

**支持类型**:
- A记录：指向IPv4地址
- AAAA记录：指向IPv6地址
- CNAME记录：指向另一个域名
- MX记录：邮件交换记录
- TXT记录：文本记录
- NS记录：名称服务器记录

### 5. DNS解析多久生效？

**生效时间**:
- 一般需要几分钟到几小时
- 取决于TTL设置
- 不同DNS服务商可能不同
- 可使用dig命令验证

### 6. 如何验证域名解析？

**验证方法**:
1. 使用系统提供的解析验证功能
2. 使用命令行工具（dig/nslookup）
3. 在线DNS查询工具
4. 检查网站访问状态

### 7. 域名解绑后还能重新绑定吗？

**回答**: 可以。域名解绑后可以重新绑定到相同或不同的网站。

### 8. 如何删除DNS记录？

**删除步骤**:
1. 在DNS记录列表找到目标记录
2. 点击"删除"按钮
3. 确认删除操作
4. 验证记录已删除

### 9. 一个网站可以绑定多少域名？

**限制说明**:
- 默认支持多个域名绑定
- 具体数量取决于系统配置
- 建议合理规划域名使用

### 10. 域名解析状态显示异常怎么办？

**排查步骤**:
1. 检查DNS服务器配置
2. 验证网络连接
3. 清除DNS缓存
4. 联系DNS服务商
