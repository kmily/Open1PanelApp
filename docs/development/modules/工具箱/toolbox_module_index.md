# 工具箱模块索引

## 模块定位

工具箱模块是Open1PanelApp的P2工具类模块，提供服务器运维所需的各类辅助工具管理能力，包括病毒扫描、入侵防御、FTP服务、磁盘管理等。

## 子模块结构

| 子模块 | 端点数 | API客户端 | 说明 |
|--------|--------|-----------|------|
| **ClamAV病毒扫描** | 12 | toolbox_v2.dart | 病毒扫描和隔离管理 |
| **FTP服务** | 8 | toolbox_v2.dart | FTP账户和配置管理 |
| **Fail2ban入侵防御** | 7 | toolbox_v2.dart | IP封禁和规则管理 |
| **主机工具** | 7 | toolbox_v2.dart | 系统诊断和维护工具 |
| **磁盘管理** | 4 | toolbox_v2.dart | 磁盘分区和挂载管理 |

## 后续规划方向

### 短期目标
- 完善各子模块的基础UI页面
- 实现工具状态监控
- 添加操作日志记录

### 中期目标
- 统一工具配置管理界面
- 实现工具健康检查
- 添加告警通知功能

### 长期目标
- 支持更多运维工具集成
- 提供工具编排能力
- 实现自动化运维场景

## 与其他模块的关系

- **主机管理**: 共享系统资源信息
- **监控管理**: 提供工具运行状态
- **日志管理**: 记录工具操作日志
- **防火墙管理**: 与Fail2ban联动
