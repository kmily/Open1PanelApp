# 任务日志模块FAQ

## 常见问题

### 1. 如何查看任务执行日志？

**查看方法**:
1. 进入任务日志管理页面
2. 浏览日志列表
3. 使用筛选功能查找特定日志
4. 点击查看详情

### 2. 日志保留多久？

**保留策略**:
- 默认保留30天
- 可在系统设置中调整
- 支持手动清理

### 3. 如何搜索特定任务的日志？

**搜索方法**:
1. 使用任务名称筛选
2. 使用时间范围筛选
3. 使用执行状态筛选
4. 组合多个条件搜索

### 4. 日志显示执行失败怎么办？

**排查步骤**:
1. 查看日志详情
2. 检查错误信息
3. 分析失败原因
4. 修复问题后重试

### 5. 如何导出任务日志？

**导出方法**:
1. 在日志列表页面
2. 选择需要导出的日志
3. 点击"导出"按钮
4. 选择导出格式
5. 下载导出文件

### 6. 日志内容过大无法显示怎么办？

**处理方法**:
1. 下载日志文件本地查看
2. 使用日志搜索功能
3. 分段查看日志内容

### 7. 如何清理历史日志？

**清理方法**:
1. 进入日志管理页面
2. 选择需要清理的日志
3. 点击"清理"按钮
4. 确认清理操作

### 8. 日志中的时间是什么时区？

**说明**: 日志时间使用服务器本地时区，与系统设置保持一致。

### 9. 如何查看任务的执行参数？

**查看方法**:
1. 进入日志详情页面
2. 查看"执行参数"部分
3. 显示任务的配置参数

### 10. 日志可以删除吗？

**说明**: 
- 可以手动删除日志记录
- 删除后无法恢复
- 建议保留重要日志
