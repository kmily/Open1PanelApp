# 网站SSL证书模块FAQ

## 常见问题

### 1. 如何申请免费SSL证书？

**申请步骤**:
1. 进入SSL证书管理页面
2. 点击"申请证书"
3. 选择"免费证书"类型
4. 输入域名和邮箱
5. 选择DNS验证方式
6. 配置DNS记录
7. 提交申请
8. 等待验证通过

### 2. 证书申请失败怎么办？

**排查步骤**:
1. 检查DNS配置是否正确
2. 验证域名解析是否生效
3. 检查邮箱是否可以接收邮件
4. 查看申请日志获取详细错误

### 3. 如何续期SSL证书？

**续期方法**:
1. 在证书列表找到目标证书
2. 点击"续期"按钮
3. 确认续期信息
4. 执行续期操作
5. 验证证书已更新

### 4. 证书快过期了怎么办？

**处理方法**:
1. 立即申请新证书
2. 配置自动续期功能
3. 临时使用HTTP访问（不推荐）
4. 通知用户证书即将过期

### 5. 如何导入CA证书？

**导入步骤**:
1. 进入CA证书管理页面
2. 点击"导入证书"
3. 选择证书文件
4. 输入证书密码（如有）
5. 导入证书

### 6. 如何配置ACME自动申请？

**配置步骤**:
1. 进入ACME配置页面
2. 添加ACME账户
3. 配置ACME服务器
4. 设置自动续期
5. 测试ACME连接

### 7. 证书验证方式有哪些？

**验证类型**:
- DNS验证：配置DNS TXT记录
- HTTP验证：在网站根目录放置验证文件
- 邮箱验证：通过邮箱点击验证链接

### 8. 如何删除SSL证书？

**删除步骤**:
1. 在证书列表找到目标证书
2. 点击"删除"按钮
3. 确认删除操作
4. 验证证书已删除

### 9. 支持哪些证书类型？

**支持类型**:
- Let's Encrypt（免费）
- 商业证书（付费）
- 自签名证书
- CA证书

### 10. 证书到期会自动续期吗？

**续期策略**:
- 需要手动配置自动续期
- 可设置提前续期天数
- 续期前会发送通知
- 支持批量续期
