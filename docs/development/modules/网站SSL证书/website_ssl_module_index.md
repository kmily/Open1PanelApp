# 网站SSL证书模块索引

## 模块定位

网站SSL证书模块是Open1PanelApp的P1高价值扩展模块，提供网站SSL证书的完整管理能力，包括证书申请、续期、CA管理、ACME配置等，是网站安全运行的重要保障。

## 子模块结构

| 子模块 | 端点数 | API客户端 | 说明 |
|--------|--------|-----------|------|
| **SSL证书管理** | 11 | ssl_v2.dart | 证书列表、申请、续期、删除 |
| **CA证书管理** | 7 | ssl_v2.dart | CA证书导入、导出、信任管理 |
| **ACME配置** | 4 | ssl_v2.dart | ACME协议配置、账户管理 |
| **HTTPS配置** | 2 | ssl_v2.dart | HTTPS强制跳转、证书绑定 |
