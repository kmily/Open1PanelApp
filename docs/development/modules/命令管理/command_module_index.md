# 命令管理模块索引

## 模块定位

命令管理模块是Open1PanelApp的P1高价值扩展模块，提供命令行操作和脚本库管理能力，支持快速执行常用命令、管理脚本模板，是运维人员日常操作的重要工具。

## 子模块结构

| 子模块 | 端点数 | API客户端 | 说明 |
|--------|--------|-----------|------|
| **命令执行** | 5 | command_v2.dart | 快速命令执行和历史记录 |
| **脚本库** | 5 | command_v2.dart | 脚本模板管理和执行 |
| **命令历史** | 3 | command_v2.dart | 执行历史查看和管理 |

## 后续规划方向

### 短期目标
- 完善命令执行界面
- 实现脚本库管理
- 添加命令历史记录

### 中期目标
- 支持命令收藏功能
- 实现脚本版本管理
- 添加命令模板库

### 长期目标
- 支持批量命令执行
- 实现命令编排功能
- 提供命令审计日志

## 与其他模块的关系

- **终端管理**: 命令执行联动
- **计划任务**: 脚本调度执行
- **日志管理**: 命令执行日志
- **主机管理**: 多主机命令执行
