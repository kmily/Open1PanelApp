# 网站配置管理模块索引

## 模块定位

网站配置管理模块是Open1PanelApp的P1高价值扩展模块，提供网站服务器配置的管理能力，包括Nginx配置、PHP扩展管理等，是网站性能优化和功能扩展的重要模块。

## 子模块结构

| 子模块 | 端点数 | API客户端 | 说明 |
|--------|--------|-----------|------|
| **Nginx配置** | 4 | openresty_v2.dart | Nginx服务器配置管理 |
| **PHP扩展** | 4 | openresty_v2.dart | PHP扩展安装和管理 |
| **PHP版本** | 1 | openresty_v2.dart | PHP版本切换管理 |

## 与其他模块的关系

- **网站管理**: 网站配置关联
- **OpenResty管理**: 服务器配置联动
- **日志管理**: 配置变更日志
- **监控管理**: 配置性能监控
