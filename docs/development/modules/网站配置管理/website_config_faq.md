# 网站配置管理模块FAQ

## 常见问题

### 1. 如何修改Nginx配置？

**修改步骤**:
1. 进入网站配置管理页面
2. 点击"Nginx配置"标签
3. 编辑配置内容
4. 点击"验证配置"
5. 验证通过后保存
6. 重载Nginx服务

### 2. Nginx配置验证失败怎么办？

**排查步骤**:
1. 查看详细错误信息
2. 检查配置语法
3. 检查配置路径
4. 参考Nginx官方文档
5. 恢复备份配置

### 3. 如何安装PHP扩展？

**安装步骤**:
1. 进入PHP扩展管理页面
2. 查看可用扩展列表
3. 选择需要安装的扩展
4. 点击"安装"按钮
5. 等待安装完成
6. 验证扩展状态

### 4. PHP扩展安装失败怎么办？

**排查步骤**:
1. 检查系统依赖
2. 查看安装日志
3. 检查PHP版本兼容性
4. 尝试手动安装
5. 联系技术支持

### 5. 如何切换PHP版本？

**切换步骤**:
1. 进入网站配置页面
2. 找到PHP版本选项
3. 选择目标版本
4. 确认切换操作
5. 验证网站运行状态

### 6. 配置修改后网站无法访问怎么办？

**处理步骤**:
1. 检查Nginx错误日志
2. 恢复备份配置
3. 重启Nginx服务
4. 验证配置语法
5. 检查端口监听状态

### 7. 如何备份Nginx配置？

**备份方法**:
1. 进入Nginx配置页面
2. 点击"备份配置"按钮
3. 输入备份描述
4. 确认备份操作
5. 下载备份文件（可选）

### 8. 支持哪些PHP扩展？

**常用扩展**:
- mysql/mysqli
- gd
- curl
- mbstring
- xml
- json
- zip
- redis
- memcached

### 9. 如何查看当前PHP版本？

**查看方法**:
1. 进入网站配置页面
2. 查看PHP版本显示
3. 或使用phpinfo()查看

### 10. 配置修改需要重启服务吗？

**说明**:
- Nginx配置修改后需要重载
- PHP配置修改后需要重启PHP-FPM
- 系统会自动提示需要重启的服务

## 故障排查指南

### Nginx配置问题
```
错误: Nginx配置验证失败
排查:
1. 检查配置语法
2. 验证配置路径
3. 检查端口占用
4. 查看Nginx日志
```

### PHP扩展问题
```
错误: PHP扩展安装失败
排查:
1. 检查系统依赖
2. 验证PHP版本
3. 检查扩展源
4. 查看安装日志
```
