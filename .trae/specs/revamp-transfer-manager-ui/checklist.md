# Checklist

## 打开能力
- [ ] Android 点击“打开文件”可成功唤起对应应用
- [ ] Android “打开下载目录”可用或提供清晰降级（复制路径）
- [x] 桌面平台“打开位置”能力不回退

## 信息架构
- [x] “下载/上传”是一级分组（窄屏可切换，宽屏双栏）
- [x] “进行中+等待中”合并展示，状态用 Section+Chip 标记
- [x] “failed + progress==100” 按完成态展示且可操作

## 交互与文案
- [x] 所有按钮文案与行为一致（重试/暂停/继续/删除/打开）
- [x] UI 无硬编码字符串（全部走 `context.l10n`）
- [x] 提示反馈使用 MD3 浮动 SnackBar（带 action/close）

## 质量验证
- [x] `flutter analyze lib` 无错误
- [ ] 关键路径手测：下载完成→打开文件；打开失败→提示+复制路径；重试→任务正确更新
